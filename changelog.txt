v0.2
------

WTemplate
- Parser recodé (réglage de nombreux bugs)
- Gestion des short tags (<?xml)
- Correction d'un bug avec les chemins de compilation qui contiennent des caractères "."
- Fixed a bug when a tag {} was present at the begining of a file
- New node: {for $key, $value in $array}
- Modified node: {include href}
- Deleted node {block} and method WTemplate::assignBlockVars no longer exists

WMain
- Template configuration déplacée directement dans WView

WRoute
- Correction d'un bug de routage lorsque wity se trouve à la racine

WView
- Special theme "_blank" added
- Method "setTpl" renamed into "setResponse"
- New method "findResponse" from WController::findTpl()

Apps
- User : correction du bug de redirection à la connexion

WNote
- Class moved to %SYS_DIR%/WCore/
- WException removed
- Removed handler session: all notes are saved into session and are automaticly treated by WView
