see \global\http://ckeditor.com

endcode

Creates a %CKEditor instance.

editor(\$name $name, \$value $value, \$config $config, \$events $events) 

In incompatible browsers %CKEditor will downgrade to plain HTML <textarea> element.

code $CKEditor = new CKEditor(); $CKEditor->editor("field1", "<p>Initial value.</p>");
endcode Advanced example:
code $CKEditor = new CKEditor(); $config = array(); $config['toolbar'] = array( array( 'Source', '-', 'Bold', 'Italic', 'Underline', 'Strike' ), array( 'Image', 'Link', 'Unlink', 'Anchor' ) ); $events['instanceReady'] = 'function (ev) { alert("Loaded: " + ev.editor.name); }'; $CKEditor->editor("field1", "<p>Initial value.</p>", $config, $events);
endcode
